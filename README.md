# RMS Landing Page

Landing page tối ưu SEO cho hệ thống quản lý nhà hàng (Restaurant Management System).

## Công nghệ sử dụng

- **Astro**: Framework tối ưu cho static sites với SEO tốt
- **Tailwind CSS**: Styling
- **TypeScript**: Type safety

## Cài đặt

```bash
npm install
# hoặc
pnpm install
# hoặc
yarn install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview production build

```bash
npm run preview
```

## SEO Features

- ✅ Meta tags tối ưu
- ✅ Open Graph tags
- ✅ Twitter Card tags
- ✅ Structured Data (JSON-LD)
- ✅ Sitemap tự động
- ✅ Robots.txt
- ✅ Semantic HTML
- ✅ Performance tối ưu
